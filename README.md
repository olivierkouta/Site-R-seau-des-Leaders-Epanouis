# Réseau des Leaders Épanouis (RLE) - Site Web

Bienvenue sur le dépôt du site officiel du **Réseau des Leaders Épanouis**.

## Technologies utilisées
- HTML5 / CSS3
- JavaScript (Vanilla)
- Design Responsive

## Structure du projet
- `index.html` : Page principale
- `style.css` : Styles et animations
- `app.js` : Logique et données de l'équipe
- `Photos RLE/` : Dossier contenant les images et portraits officiels

## Déploiement
Ce site est déployé via **GitHub Pages** et **Vercel**.
